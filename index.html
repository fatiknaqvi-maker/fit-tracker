How Grading Works

Each set is scored against your full rep range (floor to ceiling), then averaged. A set that collapses well below the floor drags the grade down — no single set can hide a bad one.

Per-Exercise Grades

S
Every set at the top of the range. Ceiling reps hit on all sets. Time to add weight (clean S only).
A
Near-perfect. High across the board with no set falling below the range floor.
B
Solid session. Most sets comfortably inside the range.
C
Bottom of the range. You hit the work but sat low. Push reps up, or the weight's a touch heavy.
D
Below the range. One or more sets collapsed under the floor, or sets were skipped. Drop the weight.

Half & Assisted Reps

Reps flagged as ½ (partial range) or A (assisted) count at 50%. e.g. 8 reps with ½ flag = 4 toward grading.

Session Rank

Average of every completed exercise's rank, rounded. Skipping exercises lowers it. Hit S across the board → 👑

When Weight Goes Up

Compounds (bench, squat, OHP, etc): need two clean S's in a row before bumping.
Isolation work: one clean S = bump immediately.
"Clean" means no half or assisted reps in that session.
